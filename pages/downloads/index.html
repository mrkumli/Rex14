1) Winrar

WinRAR is a trialware file archiver utility for Windows, developed by Eugene Roshal of win.rar GmbH. It can create and view archives in RAR or ZIP file formats, and unpack numerous archive file formats. It is mainly used for creating and extracting .zip files. To enable the user to test the integrity of archives, WinRAR embeds CRC32 or BLAKE2 checksums for each file in each archive. WinRAR supports creating encrypted, multi-part and self-extracting archives.WinRAR is a Windows-only program. An Android application called "RAR for Android" is also available. Related programs include the command-line utilities "RAR" and "UNRAR" and versions for macOS, Linux, FreeBSD, Windows CE, and MS-DOS.
